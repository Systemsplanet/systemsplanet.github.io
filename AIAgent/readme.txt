Example Path (How to use)
​Get a GitHub Token:
​Go to GitHub Settings -> Developer Settings -> Personal Access Tokens (Classic).
​Generate new token -> Check repo (Full control of private repositories).
​Copy the token (starts with ghp_).
​Get an AI Key:
​Get your OpenAI key (starts with sk-) OR your MAX/Modular key.
​Prepare a Repo:
​Create a new empty repository on GitHub (e.g., my-username/my-notes).
​Run the App:
​Open index.html on your phone or desktop.
​Settings Modal appears automatically.
​GitHub Token: Paste your ghp_... token.
​GitHub Repo: Type my-username/my-notes.
​AI Key: Paste your sk_... key.
​AI Endpoint: Leave default (OpenAI) or change if using a different provider.
​Click Save & Connect.
​Chat:
​Type "Hello".
​The app saves the chat to chat_history.json in your GitHub repo.
​It sends context to the AI and displays the response.
​If you refresh the page, it reloads the history from GitHub.
